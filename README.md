# GitHub Pages fuer ShooterNotes

Diese `docs`-Struktur ist fuer eine einfache GitHub-Pages-Seite gedacht.

## Inhalt

- `index.html`: Datenschutz, Impressum und Kontakt fuer ShooterNotes
- `styles.css`: Layout und Branding fuer die Seite
- `assets/`: Logos, App-Icon und Promo-Grafik fuer die GitHub-Pages-Seite

## GitHub Pages aktivieren

1. Repo zu GitHub pushen.
2. In GitHub das Repository oeffnen.
3. `Settings` -> `Pages`.
4. Bei `Build and deployment`:
   - `Source`: `Deploy from a branch`
   - `Branch`: deine Haupt-Branch
   - `Folder`: `/docs`
5. Speichern.
6. Nach kurzer Zeit ist die Seite unter einer URL wie dieser erreichbar:
   - `https://<dein-github-name>.github.io/<repo-name>/`

## Fuer Google Play

Diese URL kannst du dann fuer folgende Felder verwenden:

- `Privacy Policy URL`
- optional auch fuer Support-/Legal-Hinweise

Wenn du spaeter moechtest, kann die Seite auch noch um eine eigene `support.html` oder eine saubere Landingpage erweitert werden.
